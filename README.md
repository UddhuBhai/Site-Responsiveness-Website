# 🌐 Site Responsiveness Website

A simple project demonstrating different techniques used to make websites responsive across various screen sizes and devices.

## 📖 About

This project showcases four popular approaches to building responsive web layouts using HTML and CSS. It serves as a beginner-friendly reference for understanding how modern websites adapt to different screen widths.

## 🚀 Features

* 📱 Media Queries
* 🎨 CSS Flexbox
* 🏗️ CSS Grid
* 🅱️ Bootstrap Framework

## 📂 Project Structure

```text
Site-Responsiveness-Website/
│
├── index.html              # Navigation page
├── media-query.html        # Responsive design using Media Queries
├── flexbox.html            # Layout using CSS Flexbox
├── grid.html               # Layout using CSS Grid
├── bootstrap.html          # Responsive layout using Bootstrap
└── README.md
```

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## 🎯 What You'll Learn

### 1. Media Queries

* Adjust layouts based on screen width
* Create mobile-friendly designs
* Change element sizes for different devices

### 2. CSS Flexbox

* One-dimensional layouts
* Flexible spacing and alignment
* Responsive distribution of elements

### 3. CSS Grid

* Two-dimensional layouts
* Grid templates and gaps
* Responsive page structures

### 4. Bootstrap

* Responsive grid system
* Pre-built components
* Faster UI development
